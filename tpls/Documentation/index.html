Installation

Technical Requirements

  • localhost xampp, wamp or any Apache with PHP and MySQL server.
  • Livesite (preferably a sub-domain for testing).
  • For Manual Installation, you need a full installation package for Joomla! 2.5.

Quickstart Installation

The Quickstart Package consists of a complete Joomla! + Template + Extensions + Sample Content, excellent for beginner users to explore back-end settings and sample content. Installing the Quickstart is just like installing Joomla!, but you will have the full demo of the template on your server. Once your environment is ready, follow this quick guide:

  • Localhost only: create a new folder in the htdocs folder for Xampp (www folder for Wamp). Example: my-website
  • Unpack/copy all files and folders of the Quickstart folder into this newly created folder or upload them into your sub-domain folder on your live site
  • Open your favorite browser and type in the address bar localhost/my-website for localhost or sub-domain.my-live-site.com for your live site sub-domain, then hit Enter.
  • Install Joomla!. There will NOT be an "Install Sample Data" button. It will be automatically installed.
  • Watch this video tutorial on how to install Joomla! Click here!

Manual Installation

Manual Installation refers to installing the template on your Joomla! powered website. This means you already have a website with content items, modules, extensions, so you just decided you want to improve the design of your website. Please note that this template requires the Gantry 4 libraries. Do not worry, everything is included in the tpl_purityfx_J25.zip file and will be automatically installed. So here's what you have to do:

  • Log-in to your Joomla! 2.5 back-end (www.your-website.com/administrator).
  • Go to Extensions menu at the top and click Extensions Manager.
  • Click Choose File and find the provided installation file tpl_purity_J25.zip, then click the Open button.
  • Click Upload & Install. The template is now installed.
  • Go to Extensions menu at the top and click Templates Manager.
  • Select the newly installed template and click the Make default button in the top-right toolbar.
  • Check the front-end now. There it is!

Customization

Customizing the template

PurityFX styling is using LESS - enhanced styling language. Actually, LESS is the same as CSS, it is just much more powerful. In order to make some styling customization you need to edit some LESS files located in the  ../templates/purityfx/less foder. I have written pure CSS in most of th the LESS files so it can be easier for you to make changes. Here are the files you might want to edit:

  • header-dark.less - The header and some other stylings specific for the Dark Theme.
  • header-light.less - The header and some other stylings specific for the Light Theme.
  • menu-dark.less - Styling for the Dark Menu
  • menu-light.less - Styling for the Light Menu
  • k2.less - The K2 Component Styling.
  • template-custom.less - The main styling file. Most of the things you will need are located here.
  • You might also need to change something in the other files but it will be in a very rare occasions.

Changing the logo

You will need to change your logo, that's for sure :) Now, you can do that very easily from the Style tab in the Template Manager. The logo defines the height of the header bar so if your logo is with different height than the PurityFX demo logo ( 55px ) you might need to adjust the logo margins in order to make it in the middle of the header. You can do that very easily:

  1. Open template.less and go to line 21. You will find the following code:
.logo-block {
    padding: 0 15px;
    margin: 7px 10px;
}

        2. You need to increase/decrease the margin pixels, so it looks like this, for example:

.logo-block {
    padding: 0 15px;
    margin: 15px 10px;
}

Also, you might need to adjust the paddings of the menu so it can be in the middle of the header:

  1. Open menu.less and go to line 90. You will find the following code:
        > li > .item {
            font-size: @menuFontSize;
            line-height: @menuLineHeight + 2;
            padding-top: 23px;
            padding-bottom: 23px;
            padding-left: 20px;
            padding-right: 20px;
            position: relative;
            text-transform: uppercase;

        2. You need to increase/decrease the padding-top and padding-bottom pixels, so it looks like this, for example:

        > li > .item {
            font-size: @menuFontSize;
            line-height: @menuLineHeight + 2;
            padding-top: 31px;
            padding-bottom: 31px;
            padding-left: 20px;
            padding-right: 20px;
            position: relative;
            text-transform: uppercase;

Basically, this is the most important you need to know about the customization.  You might also want to check the Gantry Documentation here.

Upgrade

Upgrading the template

This stands as a comprehensive and complete guide on how to deal with upgrade on any Joomla! 2.5 website on a production live site or any other testing or development environment.

  • Back-up a copy of your template (especially if you have made customizations).
  • Go to Global Configuration and enable the Offline Mode (recommended if you have a production live site, you might not want your users to see layout breaks or critical errors while files are being replaced/copied or edited).
  • Unset the template as default (Extensions menu - Template Manager - select another template and click Default).
  • Uninstall the old version of the template ( For Joomla! 2.5 -> Extensions - Extensions Manager - Manage tab - select template and click Uninstall).
  • Install the new version of the template.
  • Set the template as default again.
  • Disable Offline Mode in the Global Configuration.

Gantry Framework Upgrade

This is just a simple one click task. If there is a new version of the Gantry Framework released you will notice that in the template settings - Extensions -> Template Manager -> PurityFX. In the Overview tab you can see if you are up-to-date or if there is a newer version. Then just click on the button.

JoomFX user

If you install the Quickstart Package you will see that there are two users - the user that you have created when installing the Quickstart and JoomFX. You need to keep JoomFX because it is the owner of the demo content, mostly the K2 Items. If you are affraid of this user you can simply change its password.

If you still insist to delete it you can do it. But then you will get an error in the K2 Stuff that user with such ID cannot be loaded. Then you simply need to delete all demo K2 Items created by JoomFX. Not only to Trash them, you need to completely delete them and the error will be gone. Then just create your K2 Categories, K2 Items, ect.

White Screen after Installation!!

On some servers/hosting providers you might get a blank screen after installing the Quickstart. All you need to do is to increase the “memory_limit” variable in the php.ini file to a higher value, for example:

memory_limit = 64M

Please have in mind that you need to restart the Apache Server after editing the php.ini file.

Boxed Layout

In order to switch to Boxed Layout you just need to rename few files as described below:

  1. Rename the "template-custom.less" file to "template-custom_full.less) - root/templates/purityfx/less
  2. Rename the "index.php" file to "index_full.php" - root/templates/purityfx
  3. Rename the "gantry.config.php" file to "gantry.config_full.php" - root/templates/purityfx
  4. Rename the "template-options.xml" file to "template-options_full.xml" - root/templates/purityfx
  5. Rename the "template-custom_boxed.less" file to "template-custom.less) - root/templates/purityfx/less
  6. Rename the "index_boxed.php" file to "index.php" - root/templates/purityfx
  7. Rename the "gantry.config_boxed.php" file to "gantry.config.php" - root/templates/purityfx
  8. Rename the "template-options_boxed.xml" file to "template-options.xml" - root/templates/purityfx

That's it. Now you have the PurityFX with boxed layout. You also will find some additional options in the Template Manager now (option to change the background).

Important:   In the Boxed Layout you should not use the Revolution (Fullscreen) module. You should use the Revolution (Normal) instead. Also, if you are going to use the RS Slideshow module you need to use 1080x450 images (in the Quickstart Package the images are 1030x450 and that's why they are not filling the whole screen).
